# **App Name**: Silk Road Rides

## Core Features:

- Vehicle Catalog: Display a catalog of available vehicles with photos, descriptions, and key features.
- Booking Form: An interactive form to collect customer information: name, contact, date, and vehicle selection.
- Contact Method Selection: Allow users to choose their preferred contact method (Telegram, WhatsApp, Call) for booking confirmations and support, with a direct link to the Telegram chat: https://t.me/toureast_transport with pre-filled text.
- Adaptive Layout: Ensure the website adapts flawlessly to different screen sizes, with mobile-first optimization.

## Style Guidelines:

- Primary color: Saturated electric purple (#BE29EC) for key interactive elements like buttons and prominent accents, evoking a premium feel.
- Background color: Ultra-light gray (#F9F9FB) to provide a clean and airy web studio aesthetic.
- Accent color: A lighter, less saturated shade of purple (#E28FF7) is used for hover states and subtle UI highlights.
- Headline font: 'Inter', a modern grotesque sans-serif for massive and confident headers.
- Body font: 'Inter', a modern grotesque sans-serif for body text.
- Use minimalist icons for vehicle features and services, in electric purple. Incorporate subtle Uzbek-inspired patterns as design accents.
- Employ a card-based system with soft shadows and rounded corners (24px) to present vehicle information and services. Use ample white space to enhance visual clarity and give a feeling of airiness.
